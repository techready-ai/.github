# TechReady Code of Conduct

TechReady 조직에 기여하는 모든 구성원, 협력자, 커뮤니티 참여자는  
다음 행동 기준을 준수해야 합니다.

## 📌 우리의 약속

우리는 다음을 약속합니다:

- 서로를 존중하고 배려합니다.
- 협업하는 모든 환경에서 포용적이고 전문적으로 행동합니다.
- 차별, 괴롭힘, 모욕적인 발언 및 행동을 허용하지 않습니다.
- 열린 커뮤니케이션과 문제 해결을 장려합니다.

## 🚫 허용되지 않는 행동

- 모욕적, 차별적, 공격적 언행
- 개인 공격 또는 괴롭힘
- 악의적 코드 또는 보안 위협 유발 행위
- 비윤리적 혹은 불법적 활동
